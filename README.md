2022年横空出世的ChatGPT在全球引起了关于AI的浪潮，也将大模型的潜力展现在所有人的面前。虽然目前ChatGPT的热度有所下降，但是从全球大小企业在AI领域的资金投入、研究力量投入以及融资情况不难看出来，这一波以ChatGPT和StableDiffusion等为代表的AI浪潮的顶峰并没有真正出现。当AI浪潮顶峰出现的时候，大概率会对众多行业、岗位造成深层次冲击。

历史的齿轮已经开始转动，为了避免在下一波AI浪潮的冲击下成为炮灰，作为普通人的我们不仅不应对新鲜领域抱有抵制心理，反而应该提前尝试、提前布局。这样，在浪潮袭来的时候，才有底气和能力迎浪而上。

本资料库搜集整理了一大堆AI工具，包括平台直接可用版本和开源可私有化部署两大类，并进一步按照应用场景进行了分类。考虑到内陆用户存在的墙的问题，还增添了是否可以直接访问的标识。当然能否直连和您当前网络有很大关系，本项目的网络环境是电信宽带。如有需要，欢迎自取。如果能随手点个Star的话，那真的是非常感谢！

本项目的内容还在持续完善中。如果您对加密货币工具、数据等内容感兴趣，欢迎移步我的其他项目仓库。如果您有其他值得推荐的AI工具，欢迎留言。
